2 The Prototyper application

The Prototyper application is an application that displays commented Oz code. The Run button executes the code. The user can edit the code and click Run again to see its modifications instantly. The Prototyper proposes a list of topics of interest for QTk. The Prototyper application can be started from the OPI by:

[Prototyper]={Module.link ["x-oz://system/wp/Prototyper.ozf"]}
{Prototyper.run}

Figure 2.1: Prototyper Snapshot

All basic functionality of QTk and all available widgets are described. Playing with the QTk Prototyper is a nice way of discovering QTk.
