1.2 QHTML Prototyper application

1.2.1 The Prototyper application

The Prototyper application is an application that displays Oz code and commentary about this code. Clicking the Run button executes the Oz code. This code can be edited and the edited code can be executed on the fly. The Prototyper proposes a list of topics of interest for QHTML.

Figure 1.1.

All basic functionalities of QHTML and all avalaible widgets are described. It is a nice way of discovering QHTML to play with the Prototyper for a while.

You can check my QHTML home page to find a direct link to a currently running Prototyper.
